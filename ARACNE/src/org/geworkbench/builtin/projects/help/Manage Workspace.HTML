Manage Workspaces

At start-up, the Workspace folder is empty. The folder can be populated by loading a workspace that was saved in a previous session or by creating a new workspace during the current session. In each session, there can be only one workspace. This help section describes opening and saving workspaces. When a workspace is saved, the application saves the entire workspace hierarchy. This includes saving projects and all files and datasets associated with those projects.

Open a workspace (stored locally or remote)

1. Select File -> Open -> Workspace from the main menu.

2. When the "Open Saved Workspace" window pops up, select the workspace and click Open.

Note: Workspace files have the extension .wsp.

At any time during the course of a session, you may want to save or delete the current workspace.

Save a workspace

1. Select File -> Save -> Workspace from the main menu.

2. Then use the "Save Current Workspace" popup window that comes up to designate the directory.

3. Enter the file name.

4. Click Save.

Note: Workspace files have the extension .wsp.
