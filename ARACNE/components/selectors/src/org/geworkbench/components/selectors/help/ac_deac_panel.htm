Activate

Activated sets affect the contents displayed in the various visualization components. Most such components provide an option for imposing a set-view. When such a view takes effect, the component will only display a subset of the original data that contains those markers (or phenotypes) that are contained in activated sets ( or markers or phenotypes). An activated set is available to other caWorkbench components. Unless it is activated, a set will not be taken into consideration by other components. More than one set can be active at the same time. Perform the following steps to activate a set:

Click on the set and the selected set label will be highlighted in blue. Then perform one of the following actions:

。 Right-clicking on the set and select Activate

。 Selecting the checkbox next to the set name.

。 Through the main menu> Commands Panel > Activate Set.
