Add a Remote Data Source

This help section describes basic usage, defining and modifying remote sources and Index Service to access data. The remote sources and file types you can access are defined in the configuration settings.

Basic Usage

。 caArray: Listing of defined remote sources.

。 Go: Accesses the remote source selected.

。 Add A New Resource: Opens the Data Source Definition page that is used add remote sources.

。 Edit: Edit the remote source parameters.

。 Remote: Remote source selector.

。 When this radio button is selected, the open file dialog window is updated to include managing remote sources functionality.

To add a remote source

1 From the Main Menu, click on Open/File/Add Data Source.

2 Populate the Data Source Definition page.

。 Note: URL and Short Name are required fields.

3 Click on OK. The configuration setting is update to reflect the addition data source

Note: The configuration setting is update to reflect the addition data source. This data source will appear

To modify a remote source

1 From the Main Menu, click on Open/File.

2 Select the data source you soul like to modify from the Data Source.

3 Click Edit.

4 Modify the contents of the Data Source Definition page.

。 Note: URL and Short Name are required fields.

5 Click Update. The system will display the modified resource in the list of available remote sources. The configuration setting is update to reflect the addition data source.

To Modify Index Service

1 From the Main Menu, click on Open/File.

2 Select the Index Service Tab.

3 Modify the URL and click Update. The application then attempts to connect to the specified index service to obtain registered remote source data.

。 Note: The configuration setting is update to reflect the modified Index Service.
